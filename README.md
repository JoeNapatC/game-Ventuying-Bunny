# 🐰 Ventuying Bunny — Random Cards

เกมสุ่มการ์ดกระต่ายผจญภัย สำหรับเล่นแบบกลุ่ม รองรับ 2–8 ผู้เล่น พร้อม Animation สุ่มไพ่สุดตื่นเต้น!

## 🎮 วิธีเล่น

1. เปิด `index.html` ในเบราว์เซอร์
2. ตั้งชื่อผู้เล่น (2–8 คน) แล้วกด **เริ่มเกม!**
3. กด **🎰 สุ่มการ์ด!** เพื่อสุ่มไพ่ในตาของผู้เล่นปัจจุบัน
4. แตะหน้าจอเพื่อไปตาถัดไป
5. ดูประวัติการเล่นได้ที่ปุ่ม 📋

## 🃏 ไพ่ในเกม

| ไพ่ | แต้ม | โอกาส |
|-----|------|-------|
| 🐰 กระต่ายเดิน | 1 | 11/21 (~52%) |
| 🐰 กระต่ายนั่ง | 2 | 3/21 (~14%) |
| 🐰 กระต่ายวิ่ง | 3 | 3/21 (~14%) |
| 🥕 แครอท (Click!) | 0 | 4/21 (~19%) |

## ✨ ฟีเจอร์

- **Weighted Random** — สุ่มไพ่ตามน้ำหนักที่กำหนด
- **Shuffle Animation** — เอฟเฟกต์สับไพ่ + พลิกเปิด + Confetti
- **Turn-based** — ระบบผลัดตาอัตโนมัติ พร้อมแสดงชื่อผู้เล่น
- **Game Log** — บันทึกทุกตาลง localStorage พร้อม Export เป็น JSON
- **Responsive** — ใช้ได้ทั้งมือถือและหน้าจอใหญ่
- **Zero Dependencies** — HTML + CSS + JS ล้วน ไม่ต้องติดตั้งอะไร

## 📁 โครงสร้างไฟล์

## 🚀 เริ่มใช้งาน

ไม่ต้องติดตั้ง — เปิดไฟล์ `index.html` ในเบราว์เซอร์ได้เลย

```bash
open index.html
```
